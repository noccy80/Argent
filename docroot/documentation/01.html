Global Variables

The core Argent server exposes a few global variables to make programming modules easier. In no particular order, here they are:

  • sys
    • logger
      • stdout(message) - Writes `message` to standard output preceded by a timestamp.
      • stderr(message) - Writes `message` to standard error preceded by a timestamp.
      • now() - Returns the current date and time in string format.
    • http
      • http - The node.js http library.
      • url - The node.js url library.
    • fs - The node.js fs library.
    • path - The node.js path library.
    • handlers - An array of all handlers loaded into Argent.
  • mods - An object containing all modules loaded into Argent. For example, mods.session is the session module.
