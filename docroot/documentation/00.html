Argent Documentation

Introduction

Argent is a web application server built on Node.js. It is modular and designed with extensibility in mind.

There are three primary components to an Argent instance:

  • The core HTTP server
  • Modules
  • Handlers

The core HTTP server serves files by default. The functionality of Argent as a web server can be configured in httpd.conf. For more information on configuring the server, read the page on httpd.conf.

Modules add functionality to server-side operations by creating namespaces within the global mods variable. The functions in these namespaces will be accessible to other modules. All of the node.js framework is available to modules, and Argent provides some additional libraries for logging and working with HTTP requests and stuff.

Handlers are special, optional parts of modules which expose some of the module functions to the network. Writing a handler is the way to create custom APIs with Argent.
